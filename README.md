# Compiler Design Programs

Programs to understand compiler design concepts.


## Run Locally

Go to the project directory

```bash
  cd any-project
```

Install dependencies

```bash
main.exe
```

## Compile from source

Change directory to source and run flex
```bash
  flex <.l file>
```

Build binary
```bash
gcc -o <name-of-binary> lex.yy.c
```

Run binary
```bash
main.exe
```
